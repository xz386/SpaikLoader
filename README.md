# 🎬 SpaikLoader

Простой и мощный загрузчик видео и музыки с YouTube

---

## 🚀 Возможности

✅ Скачать видео в MP4 (до 4K)  
✅ Скачать музыку в MP3 с тегами и обложкой  
✅ Поддержка плейлистов  
✅ Выбор качества (360p / 720p / 1080p / Лучшее)  
✅ Высокая скорость скачивания  
✅ Простой и красивый интерфейс

1. Вставь ссылку на видео или плейлист  
2. Выбери формат (MP4 или MP3)  
3. Выбери качество  
4. Нажми **Скачать**
